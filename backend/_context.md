# Backend

## What this is for
FastAPI server hosting the bias-testing engine. Takes a record + model name + protected attribute, generates counterfactual variants, runs them through a pre-trained `.pkl` classifier, returns disparity report. Also exposes Gemini-powered plain-English explanation endpoint. Output consumed by `frontend/`. Hosted on Render free tier (cannot run on Appwrite Functions — too heavy for sklearn/AIF360/pickle).

## Folder contents
| Subfolder / file pattern | What it holds |
|--------------------------|---------------|
| `main.py` | FastAPI entry, CORS, route definitions (`/stress-test`, `/explain-bias`) |
| `requirements.txt` | Python deps: fastapi, uvicorn, sklearn, pandas, numpy, aif360, google-generativeai, python-multipart |
| `engine/adversarial.py` | `generate_pairs(record, attribute)` — produces counterfactual variants via PROTECTED_SWAPS dict |
| `engine/bias_scorer.py` | `score_disparity(predictions)` — computes group score gaps, bias level (LOW/MEDIUM/HIGH) |
| `engine/mitigation.py` | Suggests reweighting / fairness constraints / data augmentation fixes |
| `models/*.pkl` | Pre-trained intentionally biased classifiers (hiring, loan). Generated by training script, not committed |
| `data/*.csv` | Training datasets with injected bias |
| `data/generate_biased_models.py` | One-off training script — produces `.pkl` files in `models/` |

## Conventions
- Models: `<domain>_model.pkl` in `models/`. Loaded by path lookup `f"models/{payload['model']}_model.pkl"`. Adding a domain = add `.pkl` + extend frontend `<select>` options.
- Datasets: `<domain>_dataset.csv` in `data/`.
- Engine modules: snake_case, single responsibility per file. No cross-imports between `adversarial`/`scorer`/`mitigation` — `main.py` orchestrates.
- PROTECTED_SWAPS dict in `adversarial.py` is single source of truth for counterfactual variants. Add new attributes there.
- Bias is the **feature**, not a bug. Models are intentionally biased on skewed historical data to expose how real systems fail. Document this clearly.
- Gemini API key via env var `GEMINI_API_KEY`. Never hardcode.
- Drop `_group` and `name` columns before `model.predict_proba()` — model trained on numeric features only.
- CORS open (`allow_origins=["*"]`) for hackathon demo only. Lock down before any real deploy.
